About Me

Engineering systems

I'm a software engineer focused on building scalable, distributed systems. I design high-performance backends that handle real-world traffic — clean architecture, low latency, high availability.

My core stack is Java & Spring Boot on the backend, with Go for performance-critical services. I work extensively in system design, distributed messaging with Kafka, caching with Redis, and container orchestration on Kubernetes.

I also apply machine learning to production systems — bridging intelligent features with scalable infrastructure. Outside of engineering, I'm an anime enthusiast always chasing the next great story.
